# AIAGENTMCP Playwright Project

This project is set up with Playwright for end-to-end testing.

## Getting Started

- Install dependencies: `npm install`
- Run tests: `npx playwright test`

## Custom Test Setup
- Add your tests in the `tests/` directory.
- Example test is provided in `tests/example.spec.ts`.

For more information, see the [Playwright documentation](https://playwright.dev/).
